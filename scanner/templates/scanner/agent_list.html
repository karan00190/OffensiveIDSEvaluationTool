 What is an Agent?
An agent is a Python script (orchestrator.py) that runs on an endpoint inside the BARC lab network. It performs controlled attacks (nmap scans, DGA queries, data exfiltration simulations) and reports results back to this server. Each agent authenticates using a unique token + HMAC signature on every request.

Quick Start
After registering — run these commands on your lab endpoint:
# 1. Copy orchestrator.py to your lab machine, then register:
python orchestrator.py --register --agent-id barc-lab-01 --reg-key YOUR_REGISTRATION_KEY
# 2. Start the persistent agent:
python orchestrator.py
# 3. Or run a single scan:
python orchestrator.py --scan 192.168.1.1 --profile fast
